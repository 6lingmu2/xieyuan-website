# 蟹源 - 源自自然，助农兴乡

蟹源是一个响应式网页，旨在推广和销售螃蟹产品并支持助农事业。

## 网站功能

- **品牌故事**：展示蟹源的品牌理念和助农使命
- **生长历程**：AI可视化螃蟹成长的一生
- **产品中心**：精选好蟹产品展示与购买
- **营养与食用**：螃蟹营养价值与食用方法指南

## 技术栈

- HTML5
- CSS3
- JavaScript
- Font Awesome 图标
