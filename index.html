2-3月

受精卵孵化

水温升至10°C以上，母蟹开始产卵。每只母蟹可产20-80万粒卵，粘附在腹部腹肢上孵化。

4-5月

溞状幼体发育

经历5期溞状幼体阶段，每期约3-5天。幼体在海水中浮游生活，以浮游生物为食。

5-6月

大眼幼体变态

溞状幼体变态为大眼幼体，具备更强的游泳能力，开始向河口洄游。

6-7月

幼蟹培育

大眼幼体变态为幼蟹，进入淡水环境。开始投喂配合饲料和天然饵料。

8-9月

成蟹养殖

幼蟹经历5-7次蜕壳，体重快速增长。科学投喂，保持水质，预防病害。

10-11月

成熟收获

螃蟹性腺发育成熟，膏满黄肥。经过严格筛选，冷链配送至您的餐桌。

食用指南

蟹黄/蟹膏

母蟹为蟹黄，橙红色；公蟹为蟹膏，白色半透明。富含蛋白质、脂肪，味道最鲜美。

推荐指数：★★★★★

蟹肉

蟹身和蟹腿中的白色肉质，富含蛋白质和微量元素，口感细嫩鲜甜。

推荐指数：★★★★★

不可食用部位

蟹胃（位于蟹黄中的三角包）、蟹肠（黑色条状）、蟹腮（两侧像眉毛的软物）、蟹心（六角形白色）

请勿食用

清蒸大闸蟹

最经典的吃法，保留原汁原味

1. 螃蟹洗净，肚皮朝上放入蒸锅 2. 姜片铺在蟹身上，去腥提鲜 3. 大火蒸15-20分钟 4. 蘸姜醋汁食用

香辣蟹

川味经典，麻辣鲜香

1. 螃蟹切块，裹淀粉油炸 2. 爆香干辣椒、花椒、葱姜蒜 3. 放入螃蟹翻炒，加料酒、生抽 4. 撒香菜出锅

蟹黄豆腐

江浙名菜，鲜美滑嫩

1. 取出蟹黄，豆腐切块焯水 2. 热油爆香姜末，下蟹黄炒散 3. 加高汤、豆腐炖煮 4. 勾芡撒葱花

不宜人群

孕妇、过敏体质、痛风患者、脾胃虚寒者慎食

食用时间

避免空腹食用，不宜与柿子、浓茶、冷饮同食

死蟹禁食

螃蟹死后体内细菌快速繁殖，产生毒素，绝对不可食用

搭配建议

可搭配黄酒、姜茶，有助于驱寒暖胃

冰箱冷藏法

将螃蟹放入保鲜盒，湿毛巾覆盖，可保存2-3天。温度保持在5-8°C。

水养保存法

将螃蟹放入浅水盆中，水位不超过蟹身一半，放置阴凉处，可存活3-5天。

熟蟹保存

蒸熟后的螃蟹冷藏可保存1-2天，再次食用需彻底加热。
